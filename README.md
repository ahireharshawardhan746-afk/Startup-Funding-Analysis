## 📖 Project Overview

The **Startup Funding Analysis** project explores investment trends in the Indian startup ecosystem between **2015 and 2020** through Exploratory Data Analysis (EDA). The project focuses on cleaning, transforming, and visualizing startup funding data to uncover meaningful patterns and business insights.

Using Python and modern data analysis libraries, the analysis identifies funding trends, leading startup hubs, popular industries, investment types, and investor preferences. The findings provide a clear picture of how India's startup ecosystem evolved during the study period.

---

## 🎯 Project Objectives

- Analyze startup funding trends from 2015–2020.
- Identify the top-funded industries and startup hubs.
- Study investment types and investor behavior.
- Perform data cleaning and feature engineering.
- Build static and interactive visualizations.
- Generate business insights from the dataset.

## 💡 Key Insights

- Startup funding is highly skewed, with a few large investments contributing significantly to the total funding.
- Seed Funding is the most common investment type, reflecting strong early-stage investment activity.
- Consumer Internet and eCommerce received the highest funding.
- Bangalore, Mumbai, and New Delhi emerged as the leading startup hubs.
- Startup funding remained strong overall, with investment activity peaking in 2017 and 2019.
- Investors showed a strong preference for scalable, technology-driven sectors.
